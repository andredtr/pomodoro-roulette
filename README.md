# Pomodoro Roulette

A simple Pomodoro timer with a roulette wheel to select tasks.

## Live Demo
https://pomodo-roulette.netlify.app/

## Features

- Add tasks to the list
- Spin the wheel to select a task
- Timer for the task
- Amazing :tada: Sound effects for the wheel and timer
